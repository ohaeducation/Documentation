# Responses (Respuestas)
Las llamadas a la API REST de Oha devuelven el código de estado HTTP. Todas las llamadas a la API devuelven cuerpos de respuesta JSON que incluyen información sobre el recurso, incluido un mensaje relacionado con la respuesta. Cada solicitud de API REST devuelve un código de estado HTTP.
